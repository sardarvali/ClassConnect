# QuizResultFragment — Student views their quiz results

---

## 📁 Location
`app/src/main/java/com/syed/classconnect/ui/quiz/QuizResultFragment.kt`

---

## 🎯 What This File Does
QuizResultFragment shows a student's quiz attempt results: score, percentage, time taken, and a question-by-question breakdown showing which answers were correct/incorrect. Uses animated percentage counter and color-coded indicators.

---

## ⚙️ Key Functions
- Displays score/totalMarks with animated percentage
- Question-by-question review: shows selected answer, correct answer, and whether it was right
- Score color coding: green (≥80%), yellow (≥50%), red (<50%)
