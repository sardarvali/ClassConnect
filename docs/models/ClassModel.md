## 🎯 What This File Does
The `ClassRoom` data class represents a class/course in ClassConnect. Each classroom has a teacher, a list of enrolled students, a unique join code, and an optional schedule. This model maps to the `/classes/{classId}` Firestore collection. Without this file, there would be no way to represent class data from the database in the app.

## 🔄 Data Flow
```
Teacher creates class → CreateClassBottomSheet → ClassRepository.createClass()
    → Firestore /classes/{id} document created with auto-generated classCode
Student joins class → JoinClassBottomSheet → ClassRepository.joinClass(code)
    → studentIds array updated, user's classIds updated
Class list displayed → ClassRepository.getClassesForStudent/Teacher()
    → callbackFlow with real-time updates → ClassViewModel → ClassListFragment
```

## ⚠️ Important Notes
- The class is named `ClassRoom` (not `Class`) because `Class` is a reserved word in Kotlin/Java (`java.lang.Class`)
- `studentIds` can grow large — Firestore documents have a 1MB size limit, which allows ~40,000 student IDs
- `classCode` is always compared in uppercase: `classCode.uppercase()` during join operations
- `schedule` uses day names as keys (e.g., "Monday") — must match `Calendar.getDisplayName()` output for widget compatibility
