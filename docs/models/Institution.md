## 🎯 What This File Does
The `Institution` data class represents a school, college, or university. Admins create institutions to group users and classes together. Each institution has a unique join code that new users enter during registration. Stored at `/institutions/{institutionId}`.

```kotlin
package com.syed.classconnect.data.model
// Package: data model layer.

import com.google.firebase.Timestamp
// Timestamp: Firebase's time type.

data class Institution(
    val id: String = "",           // Firestore document ID
    val name: String = "",         // Institution name
    val adminId: String = "",      // UID of the creating admin
    val joinCode: String = "",     // 6-char alphanumeric join code (UPPERCASE)
    val createdAt: Timestamp = Timestamp.now() // Creation timestamp
)
// joinCode is generated by ValidationUtils.generateCode(6).
// Users enter this code during registration to link their account to this institution.
// The code is always compared in uppercase.
```
