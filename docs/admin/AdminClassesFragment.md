# AdminClassesFragment — Admin manages all institution classes

---

## 📁 Location
`app/src/main/java/com/syed/classconnect/ui/admin/AdminClassesFragment.kt`

---

## 🎯 What This File Does
AdminClassesFragment shows all classes in the institution regardless of teacher. Allows admin to create classes, assign teachers, and delete classes. Uses ClassRepository with institution-wide queries.

> ⚠️ This specific fragment may be integrated into the admin dashboard navigation rather than existing as a standalone file. Admin class management is accessible through the admin bottom navigation.
